## 1. 何のためのテンプレートか

### 競合状況の再確認

| 競合 | SNS 戦略 | 我々の差別化余地 |
|---|---|---|
| ナゴレコ（IG 20万） | 店舗紹介中心・写真力で勝負 | 解釈ゼロ → ここに業界視点をぶつける |
| 名古屋情報通（X） | 新店速報・羅列型 | 鮮度勝負には参戦しない |
| ナゴグル（TikTok 10万超） | 映え動画・若年層 | 動画は次フェーズ |
| 個人グルメインフルエンサー | 主観・案件多数 | 編集独立性で勝つ |

**結論**: 「店を紹介する SNS」は飽和。我々は **「業界人だから書ける解釈」を SNS に持ち込む** 唯一無二の立ち位置を取る。

### コア・コンセプト

> **店ではなく、業界の "見方" を売る。**
>
> 1投稿で「この店がなぜ予約困難なのか」「業界人だけが知る◯◯の見極め方」を解説する。
> 結果として店舗ページへの興味（→ サイト流入）が生まれる構造にする。

## 2. テンプレート 5シリーズ

### Series A: 「予約困難の理由」シリーズ

**コンセプト**: 業界人だから知っている "予約が取れない構造" を分解する。

**フォーマット**:
- Instagram: 9:16 縦画像 1〜10枚（カルーセル）
- X: 画像1枚 + 140字
- 共通: 「なぜ◯◯は予約困難なのか?」というパワーフック

**Instagram 画像構成例**（カルーセル 6枚）:
1. **表紙**: 「【予約困難の理由 #03】◯◯（店名）」店名を大きく / 業界人推薦バッジ
2. **理由 1**: 席数・回転率の事実
3. **理由 2**: 仕入れの構造（業界人視点）
4. **理由 3**: SNS 拡散の経緯
5. **編集部所感**: insiderNote の引用
6. **CTA**: 「店舗詳細は NAGOYA BITES で」+ プロフィール誘導

**X 投稿例**:
```
【予約困難の理由 #03】〇〇

業界人として 8店舗を運営してきた立場で、
なぜこの店が常に1ヶ月先まで埋まるのかを 3つの構造で解説。

▶ 席数 20席 × 2回転 = 1日40席のみ
▶ 仕入れ◯◯のため日次売上に上限
▶ 〇〇インフルエンサー 1投稿で半年予約困難に

詳細 → https://nagoya-bites.com/stores/〇〇.html

#名古屋グルメ #業界人視点
```

### Series B: 「業界人だけが知る見極め方」シリーズ

**コンセプト**: 消費者では判断しづらい "店の見極めポイント" を業界人が教える。

**例題タイトル**:
- 「初訪の焼肉屋で "ハズレない" を見抜く 5 つの所作」
- 「居酒屋のお通しで店の本気度がわかる 3 つの理由」
- 「コース 5,000円の店で原価 50% を見極める方法」
- 「個室居酒屋の "本当に静かな店" の見分け方」

**フォーマット**:
- Instagram カルーセル 5〜7枚
- X スレッド形式（ツリー4〜6投稿）

**注意点**:
- 特定店の批判にならないよう「業界の構造的特徴」を語る
- 最後に NAGOYA BITES の該当タグ（個室/接待/業界人推薦 等）に誘導

---

### Series C: 「editor_picks 解説」シリーズ

**コンセプト**: editor_picks に追加した店を、editorReason / mediaFeatures / insiderNote の3層で深掘り紹介する。

**フォーマット（Instagram 9:16 カルーセル 4枚固定）**:
1. **表紙**: 店名 + ジャンル + エリア + 「✦ 編集部推薦」バッジ
2. **editorReason**: 60〜120字を画像化
3. **mediaFeatures**: 他媒体掲載履歴を一覧化（雑誌名 + 年）
4. **insiderNote**: 業界人視点の解釈メモ + visitStatus アイコン（visited / interview / desk）+ サイト誘導

**X**:
```
【編集部推薦 #01】◯◯（店名）

[editorReason 60-120字]

掲載歴: ◯◯（2024）/ ◯◯（2025）
業界人ノート: [insiderNote 抜粋]

詳細 → https://nagoya-bites.com/stores/...
```

**運用ルール**:
- editor_picks に新規追加した店は必ず Series C で 1投稿（ISSUE-029 と連動）
- 投稿頻度の目安: 週 2〜3本（editor_picks 拡充ペースに合わせる）

---

### Series D: 「シーン別ショート提案」シリーズ

**コンセプト**: 「今夜 2人で 8,000円以内 / 名駅 / 個室」のような **具体的な利用条件** に対して、業界人が 3 店だけ即答する。

**フォーマット**:
- Instagram 9:16 単発画像（情報カード）
- X 単発投稿

**画像構成**:
- 上部: 「今夜・2人・8,000円・名駅・完全個室」のような条件タグ
- 中央: 3店舗のサムネ + 短評（30字）
- 下部: 「あなたなら、どの店?」+ サイト誘導

**運用ルール**:
- 既存特集記事（features/*.html）を再活用 → 1記事 = 5〜10 ポストに分解可能
- 投稿頻度: 週 3〜5 本

---

### Series E: 「ジャーナル切り出し」シリーズ

**コンセプト**: `journal/` の日次記事から、SNS で刺さる 1段落を切り出して投稿する。記事流入を直接狙う。

**フォーマット**:
- X: 140字の引用 + 記事リンク
- Instagram: カルーセル 3枚（表紙 → 引用 → CTA）

**運用ルール**:
- 既存 `docs/daily-posts/` の Note/IG/X 原稿を流用 → 重複投稿せず、補完として運用
- 投稿頻度: 日次（ジャーナル更新と同じペース）

## 4. ハッシュタグ規約

### Instagram（投稿あたり 10 個前後）

**必須（ブランド系 3 個）**:
- `#NAGOYABITES`
- `#名古屋グルメ`
- `#業界人視点`

**シーン系（投稿内容に応じて 3〜5 個）**:
- `#名古屋ディナー` `#名駅グルメ` `#栄グルメ`
- `#名古屋デート` `#名古屋接待` `#名古屋宴会`
- `#名古屋個室` `#名古屋誕生日`

**ジャンル系（投稿内容に応じて 1〜2 個）**:
- `#名古屋居酒屋` `#名古屋焼肉` `#名古屋イタリアン` `#名古屋和食`

### X（投稿あたり 1〜3 個）

- `#名古屋グルメ` を必ず付ける
- 文脈次第で `#業界人視点` `#NAGOYABITES`

### NG タグ（編集独立性のため使わない）

- `#PR` `#広告` `#案件` `#タイアップ` → 我々は使う必要がない（広告ゼロのため）
- `#コラボ` → 商業色が強すぎる

## 5. 編集独立性ルール（SNS 運用）

editorial-policy.html の規約を SNS にも適用。

### やる
- ✅ 編集部の解釈・業界人視点を全面に出す
- ✅ visitStatus（visited / interview / desk）を明示
- ✅ 他媒体掲載履歴は事実ベースで紹介
- ✅ 「選ばない理由」も投稿してよい（特定店の批判ではなく構造批判の形で）

### やらない
- ❌ スポンサー投稿の受理（タイアップ・PR・案件）
- ❌ 店舗からの「投稿してください」依頼の受理
- ❌ 写真の過度なフィルタ加工（料理本来の色から逸脱しない）
- ❌ 順位 / ランキング系の煽り見出し
- ❌ フォロワー水増しサービス・相互フォロー連盟

## 6. 写真・画像の扱い

### 出典別の使用可否

| 出典 | 使用可否 | 注意 |
|---|---|---|
| 店舗からの提供画像 | ⭕ | クレジット表記必須 |
| 編集部撮影画像 | ⭕ | NAGOYA BITES クレジット |
| ホットペッパー転載画像 | ❌ | og:image でも問題（ISSUE-036） |
| Instagram 公式店舗投稿の埋め込み | ⭕ | 埋め込み機能のみ、ダウンロード転載は不可 |
| Google Maps 画像 | ❌ | 利用規約違反 |
| 食べログ画像 | ❌ | 利用規約違反 |

### 自家製 og:image / SNS カードのテンプレ要件（ISSUE-036 と連動）

- サイズ: 1200 × 630（OG）/ 1080 × 1080（IG square）/ 1080 × 1920（IG story）
- 必須要素: 店名 + エリア + 業界人推薦バッジ + NAGOYA BITES ロゴ
- フォント: Noto Sans JP / 太字
- カラー: ブランドゴールド（#bfa163 想定）+ ダークベース
