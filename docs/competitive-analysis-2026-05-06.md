**作成者**: Orchestrator (CEO)

**次回再評価**: 2026-08-06（3ヶ月後）

## 0. エグゼクティブサマリー

「名古屋で飲食店を選ぶユーザーが触れる全選択肢」を 6カテゴリ・6軸で評価した結果、NAGOYA BITES の現在地は **「素材は揃っているが外向きの存在証明が決定的に不足」** と診断する。

- **最大の Moat（守るべき独自価値）**: 「現役飲食人運営 × 業界視点の解釈層（editorReason / mediaFeatures / insiderNote / visitStatus）」は、競合 30+ サイトを観察した中で **唯一無二**。タベログ・ホットペッパー・一休・ヒトサラ・ナゴレコ・大人の名古屋のいずれも持っていない構造化編集データ。
- **最大の Gap（決定的に欠けている領域）**: **SNS 存在感の不在**。ナゴレコは Instagram 20万フォロワー / TikTok のナゴグル系は 10万超 / 名古屋情報通は X で日次更新と、消費者の発見導線の半分以上は SNS に流れているのに、NAGOYA BITES の SNS アカウントが認知されていない（site: 検索でも痕跡が薄い）。
- **最大の Quality Gap**: 推薦文 84% 空白（既存 ISSUE-017）と editor_picks の 5店止まりは、業界視点という Moat を「製品として薄い」状態のまま放置している。データ拡充が Moat の体感価値を決める。
- **追わない判断（Strategic Skip）**: 食べログの匿名口コミ大量モデル / ホットペッパーのクーポン経済 / 一休の高級セグメント / OZmall の女性向け演出は **追わない**。我々が勝てない領域で消耗する必要はない。

**CEO としての判断**: 次の 3ヶ月は **CONTENT × MARKETING の連動フェーズ** に入るべき。Moat である「業界視点」を SNS と AI 引用導線の両輪で外に出していく。STRATEGY モード（マネタイズ設計）と DATA モード（editor_picks 拡充）も並走させる。

## 1. 競合カテゴリ別の現状マップ

### Cat-A: 大手ポータル・予約サイト

| サイト | 名古屋での強み | 弱点（我々から見たスキ） |
|---|---|---|
| 食べログ | 匿名口コミの圧倒的厚み・点数アルゴリズム・全国主要 KW で TOP3 常連 | 匿名ゆえの信頼性低下・広告掲載店の優遇疑惑・地域深掘り浅い |
| ホットペッパー | 予約導線・クーポン・「カップルシート特集」「大人の隠れ家特集」など演出企画が豊富 | 広告モデルゆえに掲載=良い店ではない・編集独立性ゼロ |
| Retty | 実名口コミでバランスは良いが名古屋網羅性が中途半端 | コンテンツの鮮度が低い・SNS 連動弱 |
| ヒトサラ | **「料理人掲載数 No.1」** という独自ポジション。シェフ写真と名前を全店に表示 | 料理人ベース選定で日常使いカテゴリが弱い・予約特典は限定的 |
| 一休レストラン | 高級・接待セグメント特化・PayPay 等の決済特典・シーン×席タイプ×食材の超細粒度フィルター | 高単価帯のみ・カジュアルが取れない |
| ぐるなび | ビジネス用途特集（接待・会食ガイド）・「大人のレストランガイド」サブブランド | UI/UX が古い・SEO 順位は中位に低下中 |
| OZmall | **女性向け特化**・「女子会／推し活／ママ会」の超細粒度シーン分類・サプライズ特典（ハートケーキ／メッセージプレート） | 男性ユーザー・接待・宴会需要は取れない・名古屋の店舗カバレッジ薄い |
| まとめ系（じゃらん／くふうトリップ／tabiiro／4travel／tabemaro／TravelBook） | 「2026 年最新」KW で SEO 上位を占拠・内容は他サイトの再構成 | 独自取材ゼロ・店舗データ深さなし |

**観察結果**: 「名古屋 居酒屋 おすすめ」TOP10 のうち 7枠が大手ポータル＋まとめ系で占有。NAGOYA BITES は TOP10 に 1枠も入っていない。

### Cat-B: マップ・OS 系

| サイト | 名古屋での強み | 弱点 |
|---|---|---|
| Google Maps | 圧倒的データ量・距離検索・写真量・リアルタイム混雑情報・口コミ・ローカル SEO の最終決戦場 | 「選び方」の提案がない・編集視点ゼロ |
| Apple Maps | iPhone デフォルトの存在感だがコンテンツは Yelp/食べログ依存 | 日本での独自データ薄い |
| Yahoo!ロコ | 食べログ連携・Yahoo! 検索からの導線あり | UI 古い・モバイル弱 |

**観察結果**: Google Maps は最も避けて通れない競合。ユーザーは「店名」で必ず Maps を確認する。我々の役割は「Maps の手前で選ぶ理由を作る」こと。Maps 自体に勝つ戦略は不毛だが、Maps と連動する（Google Business Profile 経由のローカル SEO・Maps へのリンク）戦略は必要。

### Cat-C: 名古屋・地域専門メディア

| サイト | 強み | 弱点 |
|---|---|---|
| **ナゴレコ**（nagoya-meshi.com） | **Instagram 20万フォロワー**・編集部 100名超・ライター実名顔写真・特集の柔軟性（パンケーキ／夜カフェ／大須／麻辣湯／韓国料理など） | スポンサード記事多数で編集独立性に疑問・店舗データ DB なし（記事ベース） |
| **WEB大人の名古屋**（otona-no-nagoya.com） | 雑誌連動・上質ブランド・「大人のとっておき／郊外／気軽な1軒」240店厳選 | 月刊更新ペース・若年層・カジュアル取れない |
| **名古屋情報通**（jouhou.nagoya） | **毎日更新・グルメ記事 3,522件**・新店オープン速報特化（記事の 50% が新店）・PR 表記なしの中立記述 | 編集視点（解釈）なし・羅列型 |
| 日刊KELLY（kelly-net.jp） | ローカル雑誌系・「2025 最新」の見出しが SEO に効く | 雑誌の延長で深さ限定 |
| 名古屋コンシェルジュ（nagoya-info.jp） | 名古屋市公式・観光客向け・公式の権威 | 観光客向けで地元利用シーン薄い |
| haraheri / michak / choon / umaimono-blog 他 | まとめ記事 SEO 上位常連・個人サイト | 独自データなし |

**観察結果**: ナゴレコと名古屋情報通は **真の競合**。両者とも我々と同じ「名古屋特化」だが、それぞれ別の武器（SNS フォロワー / 更新頻度）で勝負している。我々の差別化は「業界視点の解釈」なので、競合の武器を真似ない。

### Cat-D: SNS（プル型・プッシュ型）

| プラットフォーム | 名古屋での主要アカウント・動向 | 我々の現状 |
|---|---|---|
| Instagram | ナゴレコ @nagoya_food（20万）・名古屋グルメ探偵アスカイ・店舗公式アカウント・#名古屋グルメ #名駅ディナー の検索動線 | 公式 IG が外部から発見されない（探索用 API 接続は ISSUE-012 で進行中） |
| X / Twitter | 名古屋情報通（@nagoya_jouhou）・グルメ探偵アスカイ・速報性・予約困難店の戦況 | 公式 X が稼働していない |
| TikTok | ナゴグル（10万超）・Maru（21万）・さや（76万）・「映え」発見導線 | 進出していない |
| YouTube | 名古屋グルメ Vlog 系・店舗紹介動画 | 進出していない |
| Threads / LINE VOOM | 軽い言及程度 | 同上 |

**観察結果**: 消費者の **発見フェーズの 50% 以上が SNS** に移っている中、NAGOYA BITES は SNS チャネルが事実上ゼロ。これが最大の Gap。

### Cat-E: 動画・ブログ・口コミ

| 媒体 | 動向 |
|---|---|
| はてな・WordPress 個人ブログ | haraheri.net、michak.jp、choon.top など、まとめ記事で SEO 上位を取る個人サイトが多数 |
| note グルメ記事 | 個人飲食人や食通の note が散発的に強い |
| YouTube Vlog 系 | 名古屋グルメ巡り Vlog で 1万〜10万再生のチャンネルが複数 |

**観察結果**: 個人ブログ・note は **広告掲載に縛られない自由な視点**で書ける点が我々と近いが、個人ゆえの脆弱性（更新停止・体系性なし）がある。我々は「集団で運営される編集部」として、個人の自由度＋組織の継続性を両立できる立場。

### Cat-F: 生成 AI・新興チャネル

| チャネル | 動向 |
|---|---|
| Google AI Overviews | 検索結果上部の AI 要約。引用元は食べログ・じゃらん・ヒトサラ・くふうトリップ等の上位サイトに偏る |
| Perplexity | 出典付き回答型。引用元は SEO 上位 + 権威サイト |
| ChatGPT / Gemini / Claude | ナレッジカットオフ依存 + Web 検索結果引用。NAGOYA BITES の被引用は確認できず |

**観察結果**: AI 引用は **SEO 上位サイトを再強化する構造**になっている。AI 時代は「AI に引用される最初の名古屋サイト」を取れば独占的優位。逆に「AI に引用されない」状態が続くと、消費者の発見導線から完全に外される。

## 4. 次に打つべき手 Top 5（CEO 推奨）

優先度順。各項目に **インパクト見込み・所要工数・担当・KPI** を付記。

### 推奨アクション #1: SNS 公式アカウント開設と日次連動運用（**P1 / 最大インパクト**）

- **What**: Instagram / X 公式アカウントを開設し、日次ジャーナル（既に毎日 1本公開中）を SNS にもクロスポストする。当面は手動運用、TikTok は次フェーズ。
- **Why**: ナゴレコ IG 20万フォロワー、ナゴグル 10万超 TikTok の戦場で、我々が SNS ゼロは戦略的に致命傷。発見導線の半分以上を諦めている状態。

- **ユーザー判断要**: アカウント開設・名前・運用ルール（スポンサー受理 NG など）

### 推奨アクション #2: editor_picks を 100店規模に拡充（**P1 / Moat 強化**）

- **What**: `data/editor_picks.json` を 5店 → 100店に段階拡充。特に「あつた蓬莱軒」「山本屋本店」「まるは食堂」型の名古屋名物名店、業界内で有名だが大手ポータルでは埋もれている店、新店で業界人が注目している店を編集視点で選定。
- **Why**: 業界視点という Moat が「製品として薄い」状態。100店あれば「どのジャンルでも業界人推薦が見つかる」体感が出る。
- **インパクト**: モーダル開封時の体感価値の劇的向上、編集規約の proof としての説得力。
- **工数**: 1店あたり editorReason 60〜120字 + mediaFeatures + insiderNote の取材・記述で 30分。100店で 50時間程度。段階配信可能。
